Question-1: What are the different ways to select an element in the DOM?

Answer:- There are Several ways to select an element in the DOM given below :
1.Selecting by ID
2.Selecting by class
3.Selecting by tag Name
4.Selecting by CSS selector
5.Selecting by Body or HTML Root

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:- The innerHTML sets the HTML content inside an element. innerText sets the visible text inside an element. textContent sets the entire text content of an element , including hidden elements.

Question-3: What is event delegation in the DOM?

Answer:- Event delegation is a Javascript technique where a single event listener is added to a parent element to handle events on its child .

Question-4: What is event bubbling in the DOM?

Answer:- Event bubbling is a concept in the DOM where an event starts from the target element and then "bubbles up" through its ancestors until it reaches the document object.
